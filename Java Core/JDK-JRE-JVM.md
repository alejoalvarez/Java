## JDK - JRE - JVM

**JDK** is Composed of 3 things:
- **JRE** - Java Runtime Environment, contain the (JVM, libraries, javaw, etc)
    - **JVM** (Java Virtual Machine) => Just in Time compiler (JIT)
- Development API 
- Java compiler
